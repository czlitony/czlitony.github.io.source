# 使用方法

基于Linux

## Clone
~~~
git clone https://github.com/czlitony/czlitony.github.io.source.git
~~~

##进入czlitony.github.io.source文件夹,依次执行
~~~
cd czlitony.github.io.source
git clone https://github.com/iissnan/hexo-theme-next themes/next
npm install
~~~
